# Content Writing

## Structure
1. Hook (first sentence grabs attention)
2. Problem (what the reader struggles with)
3. Solution (how to solve it)
4. Proof (data, examples, case studies)
5. CTA (what to do next)

## Rules
- Write for scanners: short paragraphs, subheadings, bullet points
- Lead with value, not features
- Use active voice
- Include specific numbers when possible
- End sections with transitions
- Target 1000-1500 words for blog posts
- Always include a meta description (under 160 chars)

## SEO
- Primary keyword in title, first paragraph, and 2-3 subheadings
- Use related keywords naturally throughout
- Internal links to relevant content
- External links to authoritative sources
