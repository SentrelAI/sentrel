# LinkedIn prospecting

LinkedIn is for **research and light social touches**, not mass outreach.
The goal: one concrete, recent, true fact per prospect that makes the
first email obviously hand-written.

## Research workflow (per lead)

Using the `linkedin` MCP tools:

1. Pull the prospect's profile via the LinkedIn URL from the Apollo
   enrichment table.
2. Scan their last ~10 posts/comments. A hook is something **they** said
   or did in the last 60 days — a post about their pain, a new role, a
   conference talk.
3. Check shared context: mutual connections, same groups, companies in
   common. Mention only if genuinely relevant.
4. Write the hook down with its source link in the lead table. If there's
   no real hook from LinkedIn, defer to the `news-research` skill before
   giving up on personalization.

## Social touches

- Viewing a profile or following a prospect before the first email is
  fine and useful (they may recognize the name).
- Connection requests and LinkedIn DMs follow the same approval rule as
  cold email: draft them, ask first.

## Rules

- Read-only by default. No automated liking/commenting sprees — that
  pattern gets accounts restricted.
- Max 50 profile lookups per day to stay within normal usage patterns.
- Never quote a prospect's post back at them verbatim in outreach;
  paraphrase like a human who actually read it.
